# Болгарский для собеседования на гражданство

Статичное веб-приложение-разговорник для подготовки к собеседованию по болгарскому гражданству по происхождению.

## Что внутри

- 13 учебных разделов.
- 46 вопросов для тренировки собеседования.
- Карточки формата: болгарская фраза, русская учебная транскрипция, перевод.
- Постепенное усложнение: от слов и коротких фраз до развернутых ответов.
- Готовые кейсы о себе, корнях, работе, хобби, мотивации, праздниках и культуре Болгарии.
- Пробное собеседование со случайными вопросами.

## Как открыть локально

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

Затем открыть:

```text
http://127.0.0.1:4173
```

## Как добавить аудио

В карточках можно использовать mp3-файлы из папки `audio/`. Если файла нет, приложение включает браузерную озвучку.

Рекомендуемый формат:

```text
mp3, 64-96 kbps, mono
```
