# Audio Course Files

Audio is organized by module:

- `module-01` - transcription basics.
- `module-02` - pronunciation examples.
- `module-03` - greetings and basic phrases.
- `module-04` - pronouns, `sam`, nationality forms.
- `module-05` - negation, questions and particle `li`.
- `module-06` - origin and interview phrases.
- `module-07` - numbers, dates, months and time phrases.
- `module-08` - verbs, `imam / nyamam`, plans and interview phrases.

Files referenced by the site use clean lowercase names. Original generated filenames can stay in the folder as backups, but the app should point only to clean names.
