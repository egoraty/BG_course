# Audio course files

Audio is organized by course module:

- `module-02` - module 2 words and pronunciation examples.
- `module-03` - module 3 greetings and basic phrases.
- `module-04` - module 4 pronouns, `—Б—К–Љ`, nationality forms.
- `module-05` - module 5 negation, questions and `–ї–Є`.
- `module-06` - module 6 origin and interview phrases.
- `module-07` - module 7 numbers, dates, months and time phrases.
- `module-08` - module 8 verbs, `–Є–Љ–∞–Љ / –љ—П–Љ–∞–Љ`, plans and useful interview phrases.

Use clean lowercase Bulgarian filenames for files that are referenced by the site. Original generated filenames may remain in the same folder as backups.
